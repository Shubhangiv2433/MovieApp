# 🎬 Cinema - React Movie App

A modern movie search web application built using React, Tailwind CSS, and OMDb API.

## 🚀 Features

* 🔍 Search movies in real time
* ❤️ Add & remove favorites
* 🎬 Movie cards with hover animations
* 🌙 Dark modern UI
* ⚡ Fast React + Vite setup
* 📱 Responsive design
* 🔗 Watch movie links

---

## 🛠 Tech Stack

* React JS
* Vite
* Tailwind CSS
* React Router DOM
* React Hot Toast
* React Icons
* API Integration
